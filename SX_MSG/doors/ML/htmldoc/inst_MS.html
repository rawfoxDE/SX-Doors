 Call ML as MailScan:
 --------------------


 If you Login to yer SystemX, the MessageLister has a nice
 Feature to scan for new Msgs.
 If you call the MessageLister with an Argument `s`,
 only the MailScan function of the MessageLister will work.


 Example: `ml s`


 The MailScan found new Msgs for you, for `eall` and for `all`.
 If the MailScan found such Msg, you can select between
 
 -    Q ...... Quit/Abort the Mailscan
 -    N ...... No reading, but proceed scanning
 - CR/Y ...... Read the new Msgs


 If you add a command for the Mailscan to the Internal Map of
 SystemX, you can call it from MCI-Command (~SX_MS\17\0|)
 from any ansi/ascii like in BaseBull mode.

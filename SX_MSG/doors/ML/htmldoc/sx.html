 SystemX Bulletin Board System
 -----------------------------


 SystemX is a new Bulletin Board System in the Style of
 AmiExpress.


 The archive includes all needed files to run a full
 configurable Bulletin Board System. There are serveral
 utilities posted with the archives.


 SX (eSSeXX) has a lot of features:


        - Supports 7 door types:
                        - AmiExpress (XIM)         
                        - AmiExpress AREXX (AIM)    
                        - DayDream                
                        - DayDream AREXX                
                        - Paragon/MAXsBBS          
                        - Shell/Dos/STDIO               
                        - FAME         
        - full flexable Bulletin Board System
        - up to 256 Conferences
        - up to 256 Fileares per Conference
        - up to 500 MsgBases per Conference
        - Partially compatable with /X
        - XPR protocol support (xpr#?modem.library)
        - HydraCom/SModem support.
        - comfortable GUI Config
        - comfortable GUI UserEd
        - lots of internal Functions 
        - supports MCI-Commands
        - ARexx port ("SX-MCP")
        - Line and Fullscreen Message Editor with 
          fileattachment
        - easy CommandMenu creation
        - full MCI-Code support on all ansis shown by SX
        - nearly all is editable by the sysop
        - ...


 The List is long ...

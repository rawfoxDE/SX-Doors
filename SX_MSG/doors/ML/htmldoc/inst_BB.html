 Call ML as BaseBull:
 --------------------


 If you join any Conference in yer SystemX, the MessageLister
 has a nice Feature to show the Messagebases in the joined 
 Conference.


 You have to edit a new command in SX-Config -> Internal Map
 i.e. `MI`


 in Door/String/File: Doors:ML/SX_MSG
 Mainline Arguments : i


 Call the MessageLister with an Argument `i`,
 the MessageLister will only show the MsgBases in the
 current Conference and nothing more.


 Example:      Enter following line to your Join.ANS


               ~SX_MI\17\0|     ;SXFunction 17:
                                 execute internal map function
